Pokemon AI Project Overview

Introduction

This project showcases an AI that plays Pokémon Platinum autonomously by analyzing in-game visuals and making strategic decisions. It operates by capturing two images: one before and one after a move, and subtracting them to detect changes. A Convolutional Neural Network (CNN) classifies these changes to identify movement or interactions.

The AI maps out the game environment dynamically, placing walls on the map where it detects obstacles. To handle different game scenarios, it opens the game menu to check if it's in a cutscene or battle. This approach helps the AI differentiate between normal gameplay and events where normal movement isn’t possible.

Technology

This project leverages several key technologies to build and train the AI model effectively:

  • Python: The primary programming language used for building the environment and implementing the AI logic.
  • TensorFlow: A deep learning framework used to implement and train the Convolutional Neural Networks (CNN).
  • OpenCV: A library used for image processing tasks, such as detecting changes between frames.
  • NumPy & Pandas: Libraries used for handling data processing and manipulation tasks during training.
  • Game APIs: Used to interact with the Pokémon game environment for real-time data capture and control.
