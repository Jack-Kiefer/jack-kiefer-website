Chessora Project Overview

Introduction

Chessora is a game that I am currently developing that combines the strategic depth of traditional chess with the long-term team-building mechanics found in games like autobattlers. Starting with the familiar base rules of chess, Chessora gradually introduces new pieces, bounties, and anthems through a shop interface, bridging the gap between classic chess and more complex strategic experiences.

The game features elements such as bounties, which reward players with gold for achieving specific goals, and anthems, which grant all pieces of the same type unique abilities. These additions encourage players to think creatively and adapt their strategies, making each match a unique challenge. Chessora aims to make it easier for players familiar with chess to learn and enjoy the game, while introducing a new layer of complexity that keeps gameplay fresh and engaging.

Technology

This project leverages several key technologies to build and enhance the gameplay experience:

  • Unity: The primary game engine used for creating the game, providing powerful tools for rendering and game development.
  • C#: The scripting language used to implement game logic, including piece movements, game rules, and AI behavior.
  • Unity Transport: A networking framework within Unity that facilitates real-time communication between players, enabling smooth and responsive online gameplay.
