Chessora Project Overview

Introduction

Chessora is a game I am currently developing that combines the strategic depth of traditional chess with the long-term team-building mechanics found in games like autobattlers. Built to start with the familiar base rules of chess, Chessora gradually introduces new pieces, bounties, and anthems in a shop interface that modify gameplay, providing a bridge between classic chess and more complex strategic experiences.

The game features elements such as bounties, which reward players with gold for achieving specific goals, and anthems, which grant all pieces of the same type a unique ability. These additions encourage players to think creatively and adapt their strategies, making each match a unique challenge. The goal of Chessora is to make it easier for players who already know how to play chess to learn and enjoy the game while introducing a new layer of complexity that keeps the gameplay fresh and engaging.

Technology

This project leverages several key technologies to build and enhance the gameplay experience:

  • Unity: The primary game engine used for creating the game, offering powerful tools for rendering and game development.
  • C#: The scripting language used for implementing game logic, including piece movements, game rules, and AI behavior.
  • Unity Transport: A networking framework within Unity that facilitates real-time communication between players, enabling smooth and responsive online gameplay experiences.
