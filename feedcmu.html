FeedCMU Project Overview

Introduction

The FeedCMU project is a community-driven initiative designed to reduce food waste on the Carnegie Mellon University (CMU) campus. By connecting students with free leftover food from events, FeedCMU ensures that surplus food is utilized efficiently and reaches those who can benefit from it. This not only helps reduce waste but also supports the student community by providing easy access to free food resources.

Through a user-friendly platform, FeedCMU alerts students about available food, enabling them to find and collect it before it goes to waste.

Technology Stack

The FeedCMU platform is built using the following technologies:

  • Frontend: React.js for building a dynamic and responsive user interface.
  • Backend: Node.js and Express.js for server-side logic and API management.
  • Mapping: Google Maps API for displaying food locations and navigating users efficiently.
  • Database: MongoDB for storing information about food availability and user preferences.

Challenges and Learnings

  • Effectively updating the map display using React hooks to ensure real-time updates.
  • Configuring the Google Maps API for accurate indoor mapping and navigating specific locations.
  • Implementing user-friendly features that provide instant feedback and clear navigation paths.
  • Handling data efficiently to keep the application responsive and ensure quick load times.
  • Managing teamwork and task distribution effectively to meet project deadlines.

Next Steps

  1. Implement a system to automatically delete food listings that are no longer available.
  2. Introduce user authentication to restrict the platform to CMU students and faculty.
  3. Develop a detailed indoor map that guides users directly to the specific rooms with food.
  4. Explore opportunities to expand the FeedCMU initiative to other university campuses.
