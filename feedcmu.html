FeedCMU Project Overview

Introduction

Built during HackCMU 2022, FeedCMU is a community-focused project designed to minimize food waste on the CMU campus. The idea is simple: connect students with free leftover food from various campus events, making sure surplus food goes to good use. It's a win-win situationâ€”less food waste and more students getting easy access to free meals. Our user-friendly platform sends out alerts to let students know when and where free food is available, helping them grab it before it's gone to waste

Technologies

  • Frontend: React.js for building a responsive and dynamic user interface.
  • Backend: Node.js and Express.js for handling server-side logic and API management.
  • Mapping: Google Maps API to show food locations and help users navigate effortlessly.
  • Database: MongoDB for managing data about food availability and user preferences.
