Scrapyard Scramble Project Overview

Introduction

Scrapyard Scramble is a two-player card drafting game where players assume the roles of competing inventors. The objective is to build the best robot using parts from a shared scrap pile. The game offers both player vs. player mode and a challenging AI opponent with three difficulty levels, providing varied and engaging gameplay.

Players take turns drafting cards from a shared pool, strategically selecting parts to enhance their robot's attributes. The game incorporates a point system that rewards players based on their robot's speed and intelligence, adding depth to the drafting strategy.

Technology

This project leverages the following technologies to create an engaging and strategic card game experience:

  • Python: The primary programming language used to build the game's logic and mechanics.
  • Tkinter: A Python library used for creating the graphical user interface, enabling an interactive and visually appealing game experience.
  • AI Algorithms: Implemented for creating different difficulty levels in the AI opponent, providing a scalable challenge for players.
  • Agile Development: An iterative approach used to develop and refine game mechanics, ensuring balanced gameplay and smooth user experience.
