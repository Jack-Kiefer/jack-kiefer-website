FlirtFilter Project Overview

Introduction

FlirtFilter was developed as part of HackCMU 2023 and is a playful messaging app designed to make conversations more flirtatious and engaging. By using the GPT-3 API, FlirtFilter transforms the messages sent through the app, making them flirtier and more fun as they are sent. It's an entertaining way to communicate, whether you're chatting with friends or sending sweet nothings to a romantic partner.

Technology Stack

FlirtFilter utilizes the following technologies to provide a seamless and intelligent messaging experience:

  • Frontend: React.js for creating dynamic, responsive user interfaces.
  • Backend: Node.js and Express.js for handling server-side operations and API integration.
  • AI Integration: OpenAI's GPT-3 API for natural language processing and message transformation.
  • Database: MongoDB for managing user data, including preferences and message history.
